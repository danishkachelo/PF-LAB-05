# PF-LAB-05
Name: Danish Raheem
Roll No: 25k-6549
## Lab Objectives ##
To understand the concept of nested decision structures in C programming.
To implement nested if-else statements for solving complex problems.
To apply nested switch-case statements for multi-level decision making.
To learn and use different types of operators in C (arithmetic, relational, logical, bitwise).
To understand and implement the ternary (conditional) operator.
To analyze and apply operator precedence and associativity in expressions.
To use math library functions (<math.h>) for performing mathematical operations.
To improve logical thinking for handling multiple and hierarchical conditions.
To write clean, readable, and well-structured C programs.
To reduce code redundancy by organizing complex conditions efficiently.
